Настройки

  • Fundamental.txt

    Any of these can be added for extra information of the save file

    [version] - for game version

    [stage] - for current Stage

    [strange] - for total Strange quarks

    [inflaton] - for total Inflatons

    [cosmon] - for total Cosmons

    [vacuum] - for current state of Vacuum

    [galaxy] - for current Galaxies

    [universe] - for current Universes

    [dateData] - for current date

    Y - Year

    M - Month

    D - Day

    [timeData] - for current time

    H - Hour

    M - Minute

    S - Second

    'Data' will be replaced accordingly, example
    [dateD.M.Y] ⟶ 02.02.2000

    Case sensitive

  • Format type: ,
